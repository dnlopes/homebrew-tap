# dnlopes Homebrew Tap

This is a [Homebrew](https://brew.sh) tap for projects by [@dnlopes](https://github.com/dnlopes).

## Available Formulas

- `overseer` — A terminal UI application

## Usage

Tap this repository:

```bash
brew tap dnlopes/tap
```

Then install the formulas:

```bash
brew install overseer
```

Or tap and install in one command:

```bash
brew install dnlopes/tap/overseer
```

## Updating

To update to the latest version:

```bash
brew upgrade dnlopes/tap/overseer
```
